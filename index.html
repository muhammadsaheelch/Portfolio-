About Me

I Muhammed Saheel. I love the journey from my mind to an actual design. It all starts with wire framing. Sometimes, it will be in my notes, and other times, it will be in Photoshop or Figma. After that, I jump into Visual Studio code and start making HTML. I design parts one by one, from the top header to the footer. And I give style by CSS to make it beautiful. With buttons and elements, I make it a beautiful website. By giving responsive code like percentages instead of pixels and using display Flex, making a mobile version of it never bothers me. When my schedule is strict, I use Bootstrap for a sudden catch up.
